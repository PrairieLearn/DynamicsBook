Position, velocity, and acceleration

Position vectors

Two points \(A\) and \(B\) can be used to define a vector \(\boldsymbol{r}_{AB} = \overrightarrow{AB}\) from \(A\) to \(B\). We call this the relative position of \(B\) from \(A\). If we start from the origin \(O\), so we have \(\boldsymbol{r}_{OA} = \overrightarrow{OA}\), then we call this the position vector of position \(A\). When it is clear, we will write \(\boldsymbol{r}_P\) for this position vector, or sometimes even just \(\boldsymbol{r}\).

Important point: The position vector \(\boldsymbol{r}_{OP}\) of a point \(P\) depends on which origin we are using. Using a different origin will result in a different position vector for the same point.

Example: Position vectors depend on the origin.

Show two different origins with the same \(\hat\imath,\hat\jmath\) basis at both, and a point \(P\), and show the position vector relative to each origin.

Important point: We can write any position vector in any basis.

Example: Position vectors can be written in any basis.

Show two origins, one with \(\hat\imath,\hat\jmath\) basis and one with \(\hat{a},\hat{b}\) basis. Show a point \(P\) and the two position vectors. Show that either position vector can be written in terms of either basis.

Transformation of position vectors

The position vectors of a point from two different origins differ by the offset vector between the origins. That is:

\[\begin{aligned} \overrightarrow{OP} &= \overrightarrow{OO'} + \overrightarrow{O'P} \\ \boldsymbol{r}_{OP} &= \boldsymbol{r}_{OO'} + \boldsymbol{r}_{O'P}\end{aligned}\]

Example: Converting position vectors for different origins.

Show two origins and the two position vectors to a point \(P\). Show that the relative offset relation holds.

Philosophical point: Coordinates are for positions, bases are for vectors? Coordinates can be nonlinear, bases are always linear? Position vectors relate these two things? We use the term coordinates for positions and components for vectors.

Example: Pantograph.

Consider a four-bar pantogram, as in https://en.wikipedia.org/wiki/Pantograph. Take \(O\) to be the fixed point, \(T\) to be the tracing point, and \(D\) to be the drawing point. Show that \(\overrightarrow{OD} = \alpha \overrightarrow{OT}\) for some scalar \(\alpha\). Why does this mean that the pantograph works? What determines \(\alpha\).
